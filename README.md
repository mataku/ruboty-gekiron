# Ruboty::Gekiron

Roboty handler to generate conversation topic.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-gekiron'
```

And then execute:

		$ bundle

Or install it yourself as:

    $ gem install ruboty-open_esys_oruka


## Usage

```
@ruboty topic
「ふ」から始まるかわいいものは？
```
